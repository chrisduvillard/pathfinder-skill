# Support

## Getting help

Use GitHub issues for reproducible bugs, documentation problems, installation
questions, and focused feature requests.

Before opening an issue:

- Check `README.md` and `README-INSTALL.md`.
- Confirm whether you installed Pathfinder as a Claude Code plugin, Codex
  plugin, or manual skill copy.
- Include the Pathfinder version from `VERSION.md` or the plugin manifest.
- Include only redacted logs or sanitized excerpts.

## Not supported

- Debugging private repository contents that cannot be shared safely.
- Recovering leaked credentials.
- Bypassing repository approval, safety, or security controls.
- Support for modified local copies that diverge from this repository without a
  minimal reproduction.

## Security issues

Do not ask security questions in public issues when they reveal a vulnerability.
Use `SECURITY.md` and GitHub private vulnerability reporting instead.
